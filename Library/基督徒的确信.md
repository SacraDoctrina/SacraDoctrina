# 基督徒的确信

开场白：

‘确信’指的是脱离一切的怀疑

ex) 信- 相信事实(fact)
    迷信- 相信不为事实的
    不信- 不相信事实

组成：

背诵  默想  问答  默写  应用

## 第一课 得救的确信

- 背诵：

这见证就是：上帝赐给我们永生，而这永生是在他儿子里面的。那有上帝儿子的，就有生命；没有上帝儿子的，就没有生命。（约翰一书 5:11-12）

- 默想：

谁赐永生？

永生在那里？

谁能拥有永生？

谁不能拥有永生？

- 话语：

记录圣经的目的：我把这些话写给你们信奉上帝儿子之名的人，要让你们知道自己有永生。（约翰一书 5:13）
