# 基督徒的确信

开场白：
- ‘确信’指的是脱离一切的怀疑
    - 信 - 相信事实(fact)
    - 迷信 - 相信不为事实的
    - 不信 - 不相信事实

- 组成：
    - 背诵  默想  问答  默写  应用

## 第一课 得救的确信

背诵：
- 这见证就是：上帝赐给我们永生，而这永生是在他儿子里面的。那有上帝儿子的，就有生命；没有上帝儿子的，就没有生命。(约翰一书 5:11-12)

默想：
- 谁赐永生？
- 永生在那里？
- 谁能拥有永生？
- 谁不能拥有永生？

话语：
- 记录圣经的目的：我把这些话写给你们信奉上帝儿子之名的人，要让你们知道自己有永生。(约翰一书 5:13)

1. 神怎么向你显明他的爱 ? 
    - 惟有基督在我们还作罪人的时候为我们死，上帝的爱就在此向我们显明了。(罗马书 5:8)
        - 神的爱是怎样的爱 ?
            - 无条件的爱 (罗马书 5:8)
            - 牺牲的爱 (罗马书 5:8)
            - 先爱我们 (约翰一书 4:10,19)

2. 为什么记录圣经 ?
    - 但记载这些事是要使你们信耶稣是基督，是上帝的儿子，并且使你们信他，好因着他的名得生命。(约翰福音 20:31)

3. 人的罪带来了什么结果?
    - 但你们的罪孽使你们与上帝隔绝，你们的罪恶使他转脸不听你们。(以赛亚书 59:2)

4. 为了达到得救的目的，人所付出的努力有用吗?
    - 你们得救是本乎恩，也因着信；这并不是出于自己，而是上帝所赐的；也不是出于行为，免得有人自夸。(以弗所书 2:8-9)
        - 善行：他救了我们，并不是因我们自己所行的义，而是照他的怜悯，藉着重生的洗和圣灵的更新。(提多书 3:5)
        - 宗教：除他以外，别无拯救，因为在天下人间，没有赐下别的名，我们可以靠着得救。(使徒行转 4:12)
        - 道德：有一条路，人以为正，至终成为死亡之路。(箴言 14:12)

5. 神为了引人到他面前作了什么? 
    - 因为基督也曾一次为罪受苦，就是义的代替不义的，为要引领你们到上帝面前。在肉体里，他被治死；但在灵里，他复活了。(彼得前书 3:18)

6. 人怎么得到救恩这个礼物? 
    - 凡接纳他的，就是信他名的人，他就赐他们权柄作上帝的儿女。(约翰福音 1:12)

7. 根据约翰福音 5:24 听神的话，又信的人得到哪三个结果?
    - 我实实在在地告诉你们，那听我话又信差我来那位的，**就有永生(现在)**，**不至于被定罪(未来)**，而是**已经出死入生(过去)**了。 

8. 在约翰福音 10:27-29 中耶稣向跟随自己的人有什么应许?
    - 27 我的羊听我的声音，我认识它们，它们也跟从我。 28 并且，我**赐给他们永生**；他们**永不灭亡**，谁也不能从我手里把他们夺去。 29 我父所赐给我的比万有都大，**谁也不能从我父手里把他们夺去**。

9. 人成为基督徒会有什么样的变化?
    - 所以，若有人在基督里，他就是**新造的人**：旧事已过，都变成新的了。

接受祷告：
> 耶稣，我是罪人，我愿意信耶稣得救恩。
> 我相信耶稣代替我，钉十字架受死并复活。
> 现在我打开我的心，求你进到我的里面。
> 用宝血洗净我的罪，作我生命的主带领我。
> 奉耶稣的名祷告。阿们

提问确认：
- 现在耶稣在哪里？: 在我的里面！

下面的内容中在你的生活中经历到的变化有哪些？(在适当的栏里作记号)
- 心里有平安 (  )
- 对罪有新的认识 (  )
- 从罪得胜 (  )
- 对神有新的爱 (  )
- 想读圣经了 (  )
- 态度有变化 (  )
- 对别人有关心了 (  )

背诵默写：
- 这见证就是：上帝赐给我们永生，而这永生是在他儿子里面的。那有上帝儿子的，就有生命；没有上帝儿子的，就没有生命。（约翰一书 5:11-12）

应用:
- 你怎么知道你已经有永生了？

复习：

记住：
- 接受了耶稣应该健康的成长
- 像妈妈的怀抱-教会
- 像呼吸-祷告
- 像母亲的奶–神的话语
- 运动-传福音
 
祷告
1. 感谢你让我信耶稣得救恩。让我时常心存得救的感恩之心。
2. 让我的信心健康的成长。
3. 让我也把耶稣传给别人。
