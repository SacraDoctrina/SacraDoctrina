# 내 노래는 내가 만든다

## 01 작곡 START

1. 작곡은 어떻게 시작해야 하는가?

    - What: 무엇을 만들 것인가?
    - Why: 왜 만드는 건가요?
    - When: 언제 만들기 시작해서 언제쯤 완성할 건가요?
    - Where: 어디에 쓰려고 만드는 건가요?
    - Who: 누구 연주 하거나 노래를 부를 건가요?
    - How: 어떻게 만들 건가요?

2. 작곡 방법 및 순서

    1. 작곡하기 전에 먼저 생각해 봐야 할 것들

        - 모티브(Motive) **DNA**
        - 노래의 형식(Song Form) **Climax Point**
        - 조성(Tonal) **mode**
        - 리듬 스타일(Rhythm Style or Groove) **Style**
        - 곡의 배열(Arranging Form) **Running Time**
        - 목표(Target) **Object**

    2. 작곡의 여러 가지 방법

        - Melody first approach
        - Lyric first approach
        - Harmony first approach
        - Groove first approach

3. Melody의 음역 및 종류

    1. 음역(Range)

    2. 멜로디의 종류

        - 순차진행의 멜로디(Step Motion): 기창하기에 편안한 멜로디 but 개성이 약함
        - 도약진행의 멜로디(Skip Motion): 기악적인 멜로디 개성이 강함 but 노래하기 어렵다
        - 순차 + 도약 진행의 멜로디(Step + Skip Motion): 이상적인 멜로디

        - 좋지 않은 멜로디:
            - 쉽표가 없다: 호흡을 하는 것과 같은 이치입니다.
            - 음역대를 벗어남과 너무 많은 도약: 도약은 가능하면 옥타브 이상 도약하지 않도록
            - 멜로디가 너무 없다
            - 멜로디가 너무 많다

## 02 노래의 형식

1. Song Form의 분석

    1. A 형식: 8 마디짜리
        - **포크뮤직** 즐겨 쓰던 형식
        - 멜로디가 반복되지만 반복될 때마다 다른 가사를 넣어 주로 이야기를 많이 하고픈 곡

    2. AB 형식: 8 마디짜리 + 8 마디짜리
        - 두 개의 대조되는 멜로디가 필요합니다.
        - 변형: AAB(A와 대조적인 멜로디)A, AB(A와 다른 멜로디)AC(A B와 다른 멜로디)

    3. Verse + Chorus 형식

    4. 그 밖의 다른 형식들

2. Song Form에 따른 작곡의 형태

    1. AAA 형식

    2. AB 형식

    3. AABA 형식

    4. Verse + Chorus 형식

    5. Verse + PreChorus + Chorus 형식

    6. Verse + Chorus + Bridge + Chorus 형식

    7. Verse + PreChorus + Chorus + Bridge + Chorus 형식
