# Ecclesiastes

## 传道书 1章

1. 标题 1:1
2. 传道者的言语 1:2-18
    1. 介绍传道者的言语 1:2-11
        1. 主题：虚空 1:2
        2. 理由 1:3-10
            1. 对于人生劳苦与无益处的叹息 1:3
            2. 通过对与地的比较省察人生的无常 1:4-8
                1. 命题：人生无常 1:4
                2. 观察 #1：反复无穷的自然现象 1:5-7
                    1. #1：太阳反复运转 1:5
                    2. #2：风的反复运转 1:6
                    3. #3：江河反复运转 1:7
                3. 观察 #2：万事令人厌烦 1:8
                    1. 命题：令人厌烦 1:8a
                    2. 说明：#1 说, #2 眼, #3 耳 等 1:8b
            3. 日光之下无新事 1:9-10
                1. 人生之事的反复 a 1:9a
                2. 无新事 b 1:9b
                3. 无新事 b' 1:10a
                4. 人生之事的反复 a' 1:10b
        3. 结论：所有一切无人记念 1:11
    2. 传道者根据经验的言语 1:12-18
        1. 主旨：全都是劳苦 1:12-13
            1. 背景：以色列的王-用智慧寻求究查 1:12-13a
            2. 原因（理由）：是神教人所经炼的 1:13b
            3. 结果：日光之下所有的一切都是劳苦 1:13c
        2. 陈述 #1 对于虚空的理由的言语 1:14
            1. 命题：所有一切全都是虚空 1:14
            2. 举例：弯曲的缺少的-人生的无能 1:15
        3. 陈述 #2 通过传道者经验的证词 1:16-18
            1. 背景：胜过众人的智慧与习得知识 1:16
            2. 结果：捕风捉影 1:17
        4. 结论：多有智慧 愁烦忧伤随之增多 1:18


# 传道书 2章
1. 经验的结果：享受喜乐也是虚空 2:1-11
	1. 洞察的宣言：以喜乐尝试以此相符的结果 虚空 2:1
	2. 理由：嬉笑=狂妄 喜乐=无益 2:2
	3. 对于经验的说明 2:3-10
		1. 目的：通过享乐尝试各样以为美的事 2:3
		2. 罗列成就 2:4-8
		3. 归纳：比任何人昌盛+智慧存留 2:9
		4. 归纳：没有禁止不享受的 2:10
			1. 没有禁止不享受 2:10
			2. 理由：劳碌得的分 2:11a
	4. 结果：空虚/捕风捉影/无益 2:11b
2. 领悟的理由：死亡面前的同等 2:12-23
	1. 死亡面前智慧的无用 2:12-17
		1. 背景 2:12
			1. 我观看智慧+愚昧所有事物 2:12a
			2. 毫无新事 2:12b
		2. 领悟：死亡面前等同 2:13-14
			1. 明白智慧胜过愚昧 2:13-14a
			2. 但是死亡面前没什么不同 2:14b
		3. 以告白来陈述：死亡面前的虚无 2:15
			1. 告白：反正都将死亡 2:15a
			2. 告白：虚空 2:15b
		4. 叹息 2:16
			1. 都被忘记 2:16a
			2. 智慧与愚昧在死亡面前同等 2:16b
		5. 结果 2:17
			1. 恨恶生命 2:17a
			2. 理由：日光之下所行之事=心生绝望 全都是空虚 2:17b
	2. 死亡面前劳碌的无益 2:18-23
		1. 领悟：恨恶劳碌 2:18a
		2. 理由：2:18b-19
			1. 反正会被掠走 2:18b
			2. 那人是聪明人是愚顽人尽管不知道 2:19a
			3. 但这全都将是他的 2:19b
			4. 结论：虚空 2:19c
		3. 结论：对劳碌的失望 2:20
		4. 理由：都将留给不劳碌之人 2:21a
		5. 结果：虚空/大患 2:21b
		6. 主张：一生的劳碌毫无益处 2:22-23
			1. 质疑：劳碌终将得到什么？ 2:22
			2. 理由： 2:23
				1. 日日忧虑烦恼 2:23a
				2. 夜间心也不安 2:23b
			3. 结果：虚空 2:23c
3. 结论： 所有一切都是在人能力所及之外 2:24-26
	1. 主张：吃喝享受都是虚空 2:24a
	2. 依据：出于神的手 2:24b
	3. 反问（修辞提问）: 享福谁能胜过我？ 2:25
	4. 领悟： 2:26a
	5. 结果：虚空都是捕风 2:26b
